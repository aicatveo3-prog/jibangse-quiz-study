# 지방세기본법 제1장 (총칙) — OX 학습 퀴즈

Claude Design 프로토타입(`project/지방세기본법 제1장 (총칙).dc.html`)을 **프레임워크·빌드 도구 없이** 그대로 재현한 독립 실행 정적 사이트입니다.

## 실행 방법

별도 설치가 필요 없습니다. `index.html`을 브라우저로 열기만 하면 됩니다.

- 파일을 더블클릭하거나
- 간단한 로컬 서버로 열어도 됩니다:
  ```
  cd app
  python3 -m http.server 8000
  # http://localhost:8000 접속
  ```

GitHub Pages 등 정적 호스팅에 `app/` 폴더를 그대로 올려도 동작합니다.

## 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 페이지 셸 · 폰트 · 애니메이션/탭 효과 CSS |
| `app.js` | 화면 전환·상태·이론 블록 렌더링 로직 (바닐라 JS) |
| `quizdata.js` | 문제 223개 + 파트별 이론 데이터 |

## 기능

- **홈 → 목록(파트) → 퀴즈 → 결과** 4단계 흐름
- 10개 파트 · 223문제 O/X 풀이, 답 선택 즉시 정답/해설 표시
- 기출 선지 원문 문항 + 핵심 단어를 바꾼 변형 문항을 짝으로 구성 (정답 분포 O 118 : X 105, 파트 내 같은 정답 4연속 없음)
- 파트별 **이론 정리** 패널(섹션·표·트리·⚠️/💡/예시 콜아웃) — ON/OFF 토글 상태는 브라우저에 저장(`localStorage`)
- 이론은 제1장 총칙 전체(제1절~제5절)를 퀴즈 PART 1~10 구조에 맞춰 재배치 — 각 파트 끝에 🎯 함정 체크, 필요 시 📎 심화·참고 박스
- **⭐ 최종 암기 체크리스트**(제1장 전체 총정리 25항목) — 파트 목록 상단 카드에서 한 번에 열리는 전용 화면. 항목마다 ⭕정답/❌함정 대비 표시, 탭하면 암기 완료 체크(진행률·체크 상태는 `localStorage`에 저장)
- 진행률 표시, 결과 화면 점수·다시 보기

## 참고

- 본문 글꼴은 Pretendard(CDN)이며, 오프라인에서는 시스템 글꼴로 자연스럽게 대체됩니다.
- 문항은 기출 선지 원문을 유지한 ⭕ 문항과, 앞부분은 옳고 뒷부분에서 틀리도록 핵심 단어(주체·숫자·포함/제외 등)를 바꾼 ❌ 변형 문항으로 구성했습니다.
