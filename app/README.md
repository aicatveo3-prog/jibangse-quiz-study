# 지방세 OX 학습 퀴즈 (다장 구조)

Claude Design 프로토타입(`project/지방세기본법 제1장 (총칙).dc.html`)을 **프레임워크·빌드 도구 없이** 그대로 재현한 독립 실행 정적 사이트입니다.

## 실행 방법

별도 설치가 필요 없습니다. `index.html`을 브라우저로 열기만 하면 됩니다.

- 파일을 더블클릭하거나
- 간단한 로컬 서버로 열어도 됩니다:
  ```
  cd app
  python3 -m http.server 8000
  # http://localhost:8000 접속
  ```

GitHub Pages 등 정적 호스팅에 `app/` 폴더를 그대로 올려도 동작합니다.

## 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 페이지 셸 · 폰트 · 애니메이션/탭 효과 CSS |
| `app.js` | 화면 전환·상태·이론 블록 렌더링 로직 (바닐라 JS) |
| `chapters.js` | 챕터 목록 매니페스트 (id·제목·데이터 파일·문항 수) |
| `quizdata-ch01.js` | 제1장 (총칙) — 223문항 + 이론 10파트 + 체크리스트 25항목 |
| `quizdata-ch02.js` | 제2장 (납세의무) — 238문항 + 이론 12파트 + 체크리스트 36항목 |
| `quizdata-ch03.js` | 제3장 (부과) — 111문항 + 이론 7파트 + 체크리스트 28항목 |
| `quizdata-ch04.js` | 제4장 (지방세환급금과 납세담보) — 109문항 + 이론 6파트 + 체크리스트 29항목 |

## 기능

- **홈(챕터 목록) → 목차(파트) → 퀴즈 → 결과** 흐름 — 챕터 데이터는 카드 탭 시 지연 로딩
- 총 681문제 O/X 풀이 (1장 223 · 2장 238 · 3장 111 · 4장 109), 답 선택 즉시 정답/해설 표시
- 기출 선지 원문 문항 + 핵심 단어를 바꾼 변형 문항을 짝으로 구성 (장·파트별 O/X 비율 5:5 근접, 같은 정답 4연속 없음)
- 파트별 **이론 정리** 패널(섹션·표·트리·⚠️/💡/예시 콜아웃) — ON/OFF 토글 상태는 브라우저에 저장(`localStorage`)
- 장별 이론 전체를 퀴즈 파트 구조에 맞춰 재배치 — 각 파트 끝에 🎯 함정 체크, 필요 시 📎 심화·참고 박스
- **⭐ 최종 암기 체크리스트** — 장별 전용 화면(파트 목록 상단 카드). 항목마다 ⭕정답/❌함정 대비 표시, 탭하면 암기 완료 체크. 진행 기록·체크 상태는 챕터별 `localStorage` 키에 저장
- 진행률 표시, 결과 화면 점수·다시 보기

## 참고

- 본문 글꼴은 Pretendard(CDN)이며, 오프라인에서는 시스템 글꼴로 자연스럽게 대체됩니다.
- 문항은 기출 선지 원문을 유지한 ⭕ 문항과, 앞부분은 옳고 뒷부분에서 틀리도록 핵심 단어(주체·숫자·포함/제외 등)를 바꾼 ❌ 변형 문항으로 구성했습니다.
