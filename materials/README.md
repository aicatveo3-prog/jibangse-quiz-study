# 📥 자료 업로드 폴더 (제2장 ~ 제20장)

여기는 OX 퀴즈로 변환할 **원본 자료를 붙여넣는 곳**입니다.
장(章)마다 폴더가 하나씩 있고, 폴더 안에 파일 3개가 미리 만들어져 있습니다.

```
materials/
  ch02/            ← 제2장
    문제.txt        ← 기출문제 (문제 + 선지 + 정답 + 해설 전체)
    이론정리.txt     ← 이론정리 (절·항목 구분이 있는 이론 전체)
    요점.txt        ← 요점 정리 (시험 핵심 요약 · 암기 체크리스트)
  ch03/            ← 제3장
  ...
  ch20/            ← 제20장
```

※ 제1장은 이미 완료되어 앱에 반영돼 있습니다 (`app/quizdata.js`).

## 업로드 방법 (컴퓨터 브라우저 기준)

1. 위 폴더 목록에서 올릴 장 폴더 클릭 (예: `ch02`)
2. 파일 클릭 (예: `문제.txt`)
3. 화면 오른쪽 위 **연필(✏️) 버튼** 클릭 → 편집 화면이 열림
4. 안에 있는 안내 문구를 **전부 지우고**, 메모장 내용을 **그대로 붙여넣기**
5. 오른쪽 위 초록색 **Commit changes** 버튼 클릭 → 팝업에서 다시 **Commit changes**

파일 3개(문제·이론정리·요점)를 각각 같은 방법으로 저장하면 업로드 끝.

> 💡 자료가 아주 길면: 붙여넣기 → 저장을 여러 번 나눠서 이어 붙여도 됩니다.
> 💡 파일로 갖고 있다면: 장 폴더에서 **Add file → Upload files**로 끌어다 놓아도 됩니다.
>    (이때 파일 이름을 `문제.txt` / `이론정리.txt` / `요점.txt` 로 맞춰주세요)

## 업로드 후 변환 요청

Claude 채팅에서 이렇게 요청하면 됩니다:

> **"2장 해줘"**

그러면 제1장과 동일한 기준으로 처리됩니다:

| 산출물 | 내용 |
|---|---|
| OX 퀴즈 | 선지 원문 문항 + 핵심 단어 변형 문항, O/X 5:5 분포, 기출 함정 기법 |
| 이론 정리 | 퀴즈 파트 구조에 맞춰 재배치, 파트별 🎯 함정 체크 |
| 최종 암기 체크리스트 | 요점 자료 기반, 전용 화면에 표시 |

## 자주 묻는 것

- **3개 파일을 다 채워야 하나요?** — 문제.txt만 있어도 퀴즈 변환은 가능합니다. 이론정리·요점이 있으면 이론 패널과 체크리스트까지 함께 만들어집니다.
- **순서대로 올려야 하나요?** — 아니요. 아무 장이나 준비된 것부터 올리면 됩니다.
- **모바일에서도 되나요?** — 됩니다. GitHub 모바일 웹/앱에서 같은 방법(파일 열기 → 편집 → 붙여넣기 → Commit)으로 저장할 수 있습니다.
